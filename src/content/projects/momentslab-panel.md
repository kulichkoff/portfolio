## Overview

Through a MomentsLab-powered panel in Premiere Pro, editors browse cloud media, run searches, and create subclips without breaking their editing flow.

## My Impact

I built and maintained the MomentsLab CEP extension and Angular panel used in daily Premiere workflows. Beyond feature UI, I handled CEP/Premiere constraints (panel lifecycle, host scripting, webview limits) so browse, search, and subclip features stayed stable across editor versions.
